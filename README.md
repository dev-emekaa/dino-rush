# Dino Rush 3D-Game

An endless runner dino web-game built using javascript and threejs.
![](images/dino.gif)

## Build the code

> Make sure you have node installed: https://nodejs.org/en/download/

- Install all dependencies - `npm i` 
- Run project locally - `npm start`
- Build the whole project into `/docs` folder - `npm run build`
